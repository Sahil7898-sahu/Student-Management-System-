## 🚀 Quick Setup (5 Minutes)

### Step 1: Open Hybrid System
```bash
# Open in browser
start hybrid.html
```

### Step 2: Google Sheets Setup (Optional)
1. **Go to Google Apps Script**: https://script.google.com
2. **Create New Project**
3. **Copy code from** `final-google-script.js`
4. **Paste in Code.gs**
5. **Deploy as Web App**
6. **Copy Web App URL**

### Step 3: Connect Hybrid System
1. **Open `hybrid.html`**
2. **Click "Connect Google Sheets"**
3. **Paste your Web App URL**
4. **✅ Dual storage active!**
